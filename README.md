## 学习建议

本门课是经典控制理论的线性连续系统建模与分析部分，相较于自动化原来的《自动控制理论A》少了离散系统建模和状态空间表达式分析，整体知识更加的连贯。重点是时域分析，关注系统的快准稳三大要求和性能指标。有一些思路是在电路分析中已经用过的分析方法，如时域阶跃响应和频率特性。根轨迹重点掌握绘制方法，频域响应掌握 Nyquist 图和 Bode 图的绘制以及奈奎斯特判据。本门课关注**时域-频域-复频域**的联系和转化，关注系统复频域模型和实际响应之间的关系。

如果想更深入学习控制理论，可以参考[Gaster的控制理论笔记](https://github.com/WDGaster703/ControlTheory)和[SSC的电机驱动学习笔记](https://github.com/SSC202/Engine/tree/V3.0/Note/)。
